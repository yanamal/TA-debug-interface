CSE Teaching Assistant Study

Thank you for participating in our study!
In this study, you will take on the role of a Teaching Assistant in an introductory Python programming class. You will be using a web interface to help students debug their code. On each screen, you will see the problem statement for a programming problem, and a student's incorrect solution to that problem. You will be asked to fix the student code, then write an explanation of what was wrong with it. Please do this for as many problems as you can within the time limit.
Before you begin, please answer the following questions about your background and experience.
After one hour, you will also answer a few follow-up questions about the interface you were using.
